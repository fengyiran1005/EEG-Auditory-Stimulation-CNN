# EEG-Auditory-Stimulation-CNN
The project aims to develop a model that would perform a binary classification using GRU with CNN as a baseline between EEG of individuals listening to music and individuals in a quiet resting state.
3 different GRU models were tested, namely, basic GRU, GRU with CNN front end, and GRU CNN parallel feature fusion network. GRU CNN fusion network resulted in the highest testing accuracy and AUROC value.
